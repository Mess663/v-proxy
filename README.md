# v-proxy

简单易用的 http/https 抓包工具

## 安装
```
    npm i -g v-proxy
    yarn add -g v-proxy
    pnpm i -g v-proxy
```

## 使用
```
    // 启动：
    v-proxy [port = 8899]  // or vp [port = 8899]

    port 代理端口，默认 8899

    // 关闭：
    v-proxy close  // or vp close
```
